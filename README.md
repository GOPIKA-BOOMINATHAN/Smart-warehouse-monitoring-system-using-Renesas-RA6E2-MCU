Developed an IoT-based smart warehouse monitoring system using Renesas RA6E2 MCU, DA14531 BLE module, and HS4001 temperature & humidity sensor for real-time environmental monitoring. Implemented wireless sensor communication and low-power monitoring techniques for efficient warehouse automation. Worked on embedded system integration, sensor interfacing, and real-time data acquisition for smart monitoring applications.
